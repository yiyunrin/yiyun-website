1897. Redistribute Characters to Make All Strings Equal

Problem Link

★ link

Approach

  • Traverse all strings to count the occurrences of letters.
  • Determine if the quantities of all 26 letters can be divided by n.

Complexity

  • Time complexity: O(nâˆ—m)
  • Space complexity: O(1)

Code


    class Solution:
    def makeEqual(self, words: List[str]) -> bool:
        cnt = [0 for _ in range(26)]
        n = len(words)
        for w in words:
            for c in w:
                cnt[ord(c) - ord('a')] += 1
        for i in range(26):
            if cnt[i] % n != 0:
                return False
        return True
