1155. Number of Dice Rolls With Target Sum

Problem Link

★ link

Approach

  • dp: A vector of size (n * target)
  • dp[i - 1][j - 1] represents the number of ways to achieve a total sum of j when throwing i dice
  • dp[i][j] = dp[i - 1][j - k] + ... + dp[i - 1][j - 1]

Complexity

  • Time complexity: O(n * target * k)
  • Space complexity: O(n * target)

Code


    class Solution {
    public:
        int numRollsToTarget(int n, int k, int target) {
            int mod = 1000000007;
            vector> dp(n, vector(target, 0));
            for(int i = 0;i < n;i ++){
                for(int j = i;j < min(target, (i + 1) * k);j ++){
                    if(i == 0)
                        dp[i][j] = 1;
                    else{
                        long long sum = 0;
                        for(int l = max(0, j - k);l < j;l ++)
                            sum = (sum + dp[i - 1][l])  % mod;
                        dp[i][j] = sum;
                    }
                }
            }
            return dp[n - 1][target - 1] % mod;
        }
    };
